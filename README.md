# PRCE-002 Snack Analysis

## 📌 Project Overview
This project compares the nutritional values of Starbucks and McDonald's menu items using PySpark. The goal is to analyze calorie distribution and understand patterns across different food categories.

## 🛠 Tech Stack
- Python
- PySpark
- Git
- GitHub

## 🔍 Key Insights (McDonald's Menu)

McDonald’s menu shows that calorie intensity is highest in protein-based and meal categories such as Chicken & Fish and Breakfast items. Interestingly, Smoothies & Shakes are as calorie-dense as full meals, indicating hidden sugar and fat content in beverages. In contrast, beverages and desserts tend to have lower average calories, suggesting smaller portion sizes or dilution effects.


## 📌 Final Conclusion

This project analyzed McDonald’s and Starbucks menu items using PySpark. The analysis revealed that Chicken & Fish and Breakfast items are the highest calorie categories, while beverages are the lowest. Smoothies and shakes were unexpectedly calorie-dense, showing that liquid calories can be as significant as full meals. Overall, the study highlights the importance of category-level nutritional awareness when evaluating fast food menus.
